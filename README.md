Zomato Restaurant Analysis Project
Overview

This project involves analyzing data from the Zomato restaurant platform using Microsoft Excel. The aim is to extract insights regarding restaurant ratings, locations, pricing, and customer reviews to better understand the restaurant landscape in different cities and to identify key factors influencing customer satisfaction.
Project Objectives

    Data Cleaning: Prepare the dataset for analysis by handling missing values, duplicates, and inconsistent data.
    Data Exploration: Analyze the dataset to understand the distribution of restaurants across cities, cuisines offered, pricing trends, and rating distributions.
    Visualization: Create charts and graphs to visually represent the data, making it easier to identify trends and patterns.
    Insights: Generate actionable insights based on the analysis, such as identifying popular cuisines, cities with the highest-rated restaurants, and pricing strategies.

Dataset

The dataset used for this project is the Zomato Restaurant Dataset, which includes the following columns:

    Restaurant ID: Unique identifier for each restaurant.
    Restaurant Name: Name of the restaurant.
    City: City where the restaurant is located.
    Cuisines: Types of cuisines offered by the restaurant.
    Average Cost for two: The average cost for a meal for two people.
    Currency: Currency of the country where the restaurant is located.
    Has Table booking: Whether the restaurant accepts table bookings.
    Has Online delivery: Whether the restaurant provides online delivery.
    Rating: The average rating of the restaurant.
    Votes: The number of votes the restaurant has received.

Project Workflow

    Data Import: Import the dataset into Excel.
    Data Cleaning:
        Remove any duplicate records.
        Handle missing values by either filling them with appropriate values or removing them.
        Standardize the data formats (e.g., currency).
    Data Exploration:
        Use Excel functions to calculate key statistics such as average ratings, total number of restaurants in each city, and average cost for two people.
        Filter and sort the data to identify top-rated restaurants, popular cuisines, and cities with the most restaurants.
    Data Visualization:
        Create bar charts, pie charts, and scatter plots to visualize the distribution of ratings, cuisines, and costs across different cities.
        Use pivot tables to analyze the data from different perspectives.
    Insights Generation:
        Summarize the findings from the data analysis.
        Identify trends, such as the most popular cuisines or cities with high average ratings.
        Provide recommendations for restaurant owners or potential customers based on the analysis.

Tools Used

    Microsoft Excel: For data cleaning, analysis, and visualization.
    Excel Functions: SUM, AVERAGE, COUNTIF, VLOOKUP, etc.
    Pivot Tables: To summarize and analyze the data.
    Charts and Graphs: Bar charts, pie charts, scatter plots, etc.

Results

The results of the analysis should be presented in the form of:

    Summary Tables: Showing key statistics like average ratings, most popular cuisines, etc.
    Visualizations: Charts and graphs that make it easy to understand the data at a glance.
    Insights: Key takeaways from the analysis, such as recommendations for restaurant owners or potential customers.

Conclusion

This project demonstrates how Excel can be used effectively to analyze and visualize restaurant data. By following the steps outlined above, one can gain valuable insights into the restaurant industry, which can be useful for making data-driven decisions.
